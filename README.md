# HWDE_Tracker

Tracker français **non officiel** pour *Hyrule Warriors: Definitive Edition* (Nintendo Switch).
Il couvre le mode Aventure de toutes les cartes : missions, cartes-objets, Skulltulas d'or,
clés de boss, fées et nourritures, avec localisation française intégrale.

🔗 **[Ouvrir le tracker](https://solruu.github.io/HWDE_Tracker/)**

## Crédits & sources

La majeure partie des données provient des **guides de SBAllen**, publiés sur **GameFAQs**
pour *Hyrule Warriors Legends* (3DS), puis adaptés ici pour la Definitive Edition (Switch) —
les coordonnées et certaines récompenses diffèrent entre les deux versions.

- Guides de **SBAllen** — [GameFAQs · Hyrule Warriors Legends](https://gamefaqs.gamespot.com/3ds/167257-hyrule-warriors-legends/faqs)
- Miroir des cartes — [MetroWind](https://metrowind.github.io/hyrule-warriors-maps/maps.html)

**Un grand merci à SBAllen** pour ce travail de référence, qui constitue le point de départ
de l'essentiel de ce projet.

## Nature du projet

Projet fan, non officiel et sans but lucratif. *Hyrule Warriors* et *The Legend of Zelda*
sont des marques de Nintendo / Koei Tecmo ; ce tracker n'est ni affilié ni approuvé par
ces sociétés. Les données sont fournies à titre de référence personnelle.
